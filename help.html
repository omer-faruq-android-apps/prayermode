How to Enable "Install from Unknown Sources"

Security Notice: This is a standard Android security feature. You only need to enable this temporarily to install Prayer Mode. We recommend turning it off again after installation for security.

Since Prayer mode is distributed outside the Google Play Store, you need to enable "Install unknown apps" to install it. The steps vary by Android version:

For Android 8.0 to Android 12

1 Open Settings on your device

2 Tap on Apps & notifications

3 Select Advanced > Special app access

4 Tap on Install unknown apps

5 Select the app you're using to download Prayer Mode (e.g., Chrome, your file manager)

6 Toggle Allow from this source to ON

For Android 13 and later

1 Open Settings

2 Go to Apps

3 Select Special app access

4 Tap on Install unknown apps

5 Choose your browser or file manager

6 Enable Allow from this source

For Android 7.1 and Earlier

1 Go to Settings

2 Select Security

3 Enable Unknown sources

4 Confirm any security warning

Important Notes

  • App-specific permission: On Android 8+, you enable this for the specific app you're using to install (like Chrome or your file manager), not globally
  • One-time use: You can disable this permission after installing Prayer mode
  • Security prompt: On newer Android versions, you might see a "Restricted settings" warning - this is normal
  • Device variations: Some manufacturers (Samsung, Xiaomi, etc.) may have slightly different menu names

After Installation

Once Prayer mode is installed, we recommend:

  • Disabling the "Install unknown apps" permission for the app you used
  • Granting the necessary permissions when prompted (location, notifications, DND access)
  • Setting your preferred calculation method in the app settings
